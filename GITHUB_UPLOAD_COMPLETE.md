## 📖 用户使用指南

### 新用户克隆和启动

```bash
# 1. 克隆项目
git clone https://github.com/CalamityXeroc/MapBox-Visualization.git
cd MapBox-Visualization

# 2. 安装依赖
npm install
cd backend && npm install && cd ..

# 3. 配置环境（复制示例）
cp backend/.env.example backend/.env
# 编辑 backend/.env，填入你的 TDT_KEY

# 4. 启动应用
npm run backend  # 终端 1
npm run dev      # 终端 2
```
